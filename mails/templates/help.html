What is rmd.io?

rmd.io is a service, which reminds you of unanswered e-mails and helps you to manage delegated tasks.
For example: If you send an e-mail to '5d@rmd.io', then we'll send you the e-mail back 5 days later. This also works by using 'w' for weeks and 'm' for months.

Anti-Spam

Why should I use the anti-spam measure?

The anti-spam measure feature protects you from receiving spam mails. If you enable it, you won't get any e-mails from faked senders.

What does it do?

The anti-spam measure generates a personal anti-spam key. It checks whether the key from the e-mails sender is equal to the user key. If that isn't the case, the e-mail is dropped.

How can I use it?

At first, you have to enable the anti-spam measure feature in your settings. The settings can be found at the drop-down navigation in the upper right corner.
After enabling it, you can download a custom vCard file with a subset of all possible addresses.
If you don't use an address from the address book, then you have to manually include the key into the rmd.io e-mail address (e.g. 2d.jszelaigdj@rmd.io).

Additional Addresses

What is the additional addresses feature for?

Additional addresses allow you, to add more than one address, from where you can send e-mails to rmd.io.
No need to register multiple times, but still use rmd.io for all your e-mail addresses.

How can I use it?

Go to your settings and type an additional address into the form and click save. You will receive a confirmation e-mail with a confirmation link. Please click the link to finally enable the additional address.
